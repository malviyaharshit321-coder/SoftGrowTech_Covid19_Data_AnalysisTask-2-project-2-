# Covid-19 Global Data Analysis 📊
### SoftGrowTech Data Science Internship - Project 2

## 📌 Project Overview
An exploratory data analysis (EDA) of global Covid-19 trends. This project tracks the progression of the pandemic, focusing on infection rates, recovery patterns, and mortality statistics across different countries and timeframes.

## 🛠️ Tech Stack
- **Language:** Python
- **Libraries:** Pandas, NumPy, Seaborn, Matplotlib
- **Data Source:** John Hopkins University (Aggregated Data)

## 📈 Visualizations Included
- Global Confirmed vs. Recovered Cases (Time-Series)
- Mortality Rate Analysis over time.
- Country-specific growth trends.

## 🚀 How to Run
1. Open the `Covid19_Analysis.ipynb` notebook.
2. The script automatically fetches the latest data from the dataset URL provided in the code.
3. Execute the cells to generate interactive visualizations.
